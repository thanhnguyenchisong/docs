# Example — Project minh họa Spring Data JPA

Spring Boot + Spring Data JPA: query methods, `@Query`. Chạy được để **test và học** theo tài liệu spring-jpa.

## Chạy

```bash
mvn spring-boot:run
```

- API: http://localhost:8082/api/products  
- Search: http://localhost:8082/api/products/search?q=xxx  
- H2: http://localhost:8082/h2-console

Đọc kèm [../README.md](../README.md) và các file trong backend/spring-jpa.
