# Example — Project minh họa Java

Maven project thuần Java (không Spring) để **test và học**: Collections, Stream API, Lambda.

## Chạy

```bash
mvn compile exec:java
```

Hoặc:

```bash
mvn package
java -cp target/java-example-1.0.0.jar vn.docs.example.Main
```

Đọc kèm [../README.md](../README.md) và các file `01-*.md` … trong thư mục backend/java.
