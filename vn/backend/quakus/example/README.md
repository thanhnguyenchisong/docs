# Example — Project minh họa Quarkus

Quarkus + REST (RESTEasy Reactive). Chạy được để **test và học** theo tài liệu Quarkus.

## Chạy (dev mode)

```bash
mvn quarkus:dev
```

Mở http://localhost:8080/hello

## Build JAR

```bash
mvn package
java -jar target/quarkus-app/quarkus-run.jar
```

Đọc kèm [../README.md](../README.md) và các file trong backend/quakus.
