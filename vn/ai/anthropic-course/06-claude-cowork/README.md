# Claude Cowork — Làm việc cùng Claude trên files thực

> 🔗 Khóa chính thức: [Introduction to Claude Cowork](https://anthropic.skilljar.com/introduction-to-claude-cowork)
> 📌 Level: Beginner | Dành cho tất cả mọi người | Miễn phí

---

## 📖 Giới thiệu

Cowork là Claude **làm việc trực tiếp** với files, folders, apps của bạn — đọc, sửa, và tạo output thực trên máy. Khác với Chat (hội thoại), Cowork là **working session** — bạn mô tả task, Claude lập kế hoạch và thực thi, bạn steer dọc đường.

---

## 🎯 Sau khóa học, bạn sẽ biết

- Phân biệt Chat vs Cowork
- Chạy task đầu tiên end-to-end
- Cung cấp context hiệu quả
- Setup plugins và skills
- Steer multi-step tasks
- Quản lý permissions và chọn model
