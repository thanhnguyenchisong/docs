## 📖 Giới thiệu

Claude Code là **AI coding agent** hoạt động trong terminal, VS Code, JetBrains — không chỉ trò chuyện mà **đọc code**, **chạy lệnh**, **sửa files**, và **commit** thay bạn. Khác biệt với chat-based AI tools, Claude Code là **agentic** — nó tự quyết định cần dùng tool nào để hoàn thành task.

---

## 🎯 Sau khóa học, bạn sẽ biết

- Giải thích AI coding agent là gì, khác biệt với AI chat
- Hiểu agentic loop, context window, tools, và permissions
- Cài đặt Claude Code trên terminal, VS Code, JetBrains, Desktop
- Viết prompt hiệu quả với approval mode, Plan Mode
- Áp dụng workflow: **Explore → Plan → Code → Commit**
- Quản lý context window: `/compact`, `/clear`, `/context`
- Tạo CLAUDE.md file
- Build custom subagents, kết nối MCP servers, viết hooks
