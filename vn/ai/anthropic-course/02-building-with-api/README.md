# Building with the Claude API

> 🔗 Khóa chính thức: [Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api)
> 📌 Level: Beginner → Advanced | Dành cho Developer | Miễn phí

---

## 📖 Giới thiệu

Đây là khóa học **toàn diện nhất** của Anthropic, bao gồm mọi thứ từ API cơ bản đến xây dựng hệ thống agent phức tạp. Khóa gồm 9 module chính với video, bài tập thực hành và quiz.

---

## 🎯 Sau khóa học, bạn sẽ biết

- Setup và authenticate với Anthropic API
- Xây dựng cuộc trò chuyện đơn/đa lượt (single/multi-turn)
- Cấu hình system prompts, temperature, streaming, structured output
- Thiết kế và chạy prompt evaluation workflows
- Áp dụng kỹ thuật prompt engineering: XML tags, few-shot, clear directives
- Tích hợp tool use: custom tools, web search, batch operations
- Xây dựng hệ thống RAG với chunking, embeddings, BM25
- Sử dụng extended thinking, vision, PDF, citations, prompt caching
- Phát triển MCP servers và clients
- Thiết kế agent workflows: parallelization, chaining, routing

---

## 📋 Prerequisites

- Thành thạo Python
- Hiểu cơ bản JSON
