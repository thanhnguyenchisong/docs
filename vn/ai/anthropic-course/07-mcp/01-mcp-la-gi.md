## рҹ”№ Transport Protocols

### stdio (Standard I/O)
```
Cho local servers вҖ” parent process tбәЎo child process:
Host вҶ’ spawn "python server.py" вҶ’ giao tiбәҝp qua stdin/stdout
```

### HTTP + Server-Sent Events (SSE)  
```
Cho remote servers вҖ” HTTP request/response + SSE streaming:
Client вҶ’ HTTP POST вҶ’ Server
Server вҶ’ SSE events вҶ’ Client
```

## рҹ”№ Request Flow

```
1. User: "TбәЎo GitHub issue cho bug login"
2. Host (Claude Desktop) вҶ’ gб»ӯi cho Claude AI
3. Claude AI: "TГҙi cбә§n dГ№ng tool create_issue"
4. MCP Client вҶ’ gб»ӯi tool call вҶ’ MCP Server (GitHub)
5. MCP Server вҶ’ gб»Қi GitHub API вҶ’ tбәЎo issue
6. MCP Server вҶ’ trбәЈ result вҶ’ MCP Client
7. MCP Client вҶ’ trбәЈ cho Claude AI
8. Claude AI: "ДҗГЈ tбәЎo issue #456: Bug login"
```
