# Subagents — Giới thiệu về Sub-agents trong Claude Code

> 🔗 Khóa chính thức: [Introduction to Subagents](https://anthropic.skilljar.com/introduction-to-subagents)
> 📌 Level: Intermediate | Dành cho Developer | Miễn phí

---

## 📖 Giới thiệu

Sub-agents là một trong những cách thực tế nhất để tận dụng tối đa Claude Code trong các phiên làm việc dài. Chúng cho phép bạn **ủy thác nhiệm vụ** cho các assistant riêng biệt, hoạt động trong context window độc lập, và trả về kết quả tóm gọn — giữ cho conversation chính của bạn luôn sạch và tập trung.

---

## 🎯 Sau khóa học, bạn sẽ biết

- Cách subagent hoạt động — context window riêng, input/output flow
- Tạo custom subagent bằng lệnh `/agents`
- Thiết kế subagent hiệu quả — structured output, obstacle reporting, tool limiting
- Khi nào nên dùng và khi nào KHÔNG nên dùng subagent
