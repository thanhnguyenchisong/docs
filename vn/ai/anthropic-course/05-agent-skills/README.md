# Agent Skills — Tạo Reusable Instructions cho Claude Code

> 🔗 Khóa chính thức: [Introduction to Agent Skills](https://anthropic.skilljar.com/introduction-to-agent-skills)
> 📌 Level: Intermediate | Dành cho Developer | Miễn phí

---

## 📖 Giới thiệu

Skills là **markdown instructions tái sử dụng** mà Claude tự động áp dụng đúng task, đúng lúc. Thay vì lặp lại cùng hướng dẫn mỗi lần, bạn dạy Claude **một lần** qua SKILL.md — Claude tự biết khi nào cần dùng.

---

## 🎯 Sau khóa học, bạn sẽ biết

- Skills là gì, khác gì CLAUDE.md, hooks, subagents
- Tạo SKILL.md với frontmatter, descriptions trigger matching
- Cấu hình nâng cao: multi-file skills, allowed-tools
- Chia sẻ skills qua repos, plugins, enterprise settings
- Troubleshoot skills: trigger failures, priority conflicts
