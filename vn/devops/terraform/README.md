## Hướng dẫn Terraform từ cơ bản đến chuyên gia

Đọc theo thứ tự:

1. `01-basics.md` — Terraform là gì, cài đặt, CLI fundamentals.
2. `02-workflow-and-state.md` — init/plan/apply/destroy, backends, workspaces.
3. `03-modules-and-reuse.md` — module structure, registry, composition, variables/outputs.
4. `04-testing-and-ci.md` — fmt/validate/tflint/checkov, unit/integration tests, CI flow.
5. `05-advanced-and-security.md` — locking, drift detection, secrets handling, code quality, multi-env patterns.

Sử dụng `terraform fmt/validate/plan` khi bạn đi để thực hành.
