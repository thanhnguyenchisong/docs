## Hướng dẫn nhanh Helm từ cơ bản đến master

Theo dõi các phần theo thứ tự:

1. `01-helm-basics.md` — Helm là gì, cài đặt, lệnh cốt lõi, cơ bản repos.
2. `02-chart-authoring.md` — cấu trúc chart, Chart.yaml, dependencies, hooks.
3. `03-values-and-templating.md` — values, templates, functions, patterns phổ biến.
4. `04-release-ops.md` — install/upgrade/rollback, diffs, testing, cơ bản CI.
5. `05-advanced-patterns.md` — subcharts, library charts, JSON schema, bảo mật, testing chart.

Sử dụng `helm --help` và `helm template` trong khi đọc để validate understanding.
