# Example — Jenkins Pipeline mẫu

`Jenkinsfile` mẫu: pipeline hai stage Build và Test. **Chạy được** khi tạo Pipeline job trong Jenkins trỏ tới file này.

## Cách dùng

1. Trong Jenkins: New Item → Pipeline.
2. Pipeline definition: Pipeline script from SCM (Git) hoặc paste nội dung `Jenkinsfile`.
3. Build Now — xem log từng stage.

Đọc kèm [../README.md](../README.md) và các file trong devops/jenkins.
