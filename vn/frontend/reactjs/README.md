## 📝 Yêu Cầu Trước Khi Bắt Đầu

- **HTML/CSS**: Hiểu cơ bản về HTML tags, CSS selectors, flexbox, grid
- **JavaScript ES6+**: let/const, arrow functions, destructuring, spread/rest, promises, async/await, modules
- **Git**: Cơ bản về version control
- **Terminal/Command Line**: Biết sử dụng cơ bản
- **Node.js & npm**: Đã cài đặt Node.js (v18+)

## 💡 Cách Sử Dụng Tài Liệu

1. **Đọc theo thứ tự**: Các bài được sắp xếp từ dễ đến khó
2. **Thực hành code**: Mỗi bài đều có ví dụ code, hãy tự gõ lại
3. **Làm bài tập**: Cuối mỗi bài có bài tập thực hành
4. **Xây dựng dự án**: Áp dụng kiến thức vào dự án thực tế (Phần 27)
5. **Ôn tập phỏng vấn**: Sử dụng phần 28 để chuẩn bị phỏng vấn
