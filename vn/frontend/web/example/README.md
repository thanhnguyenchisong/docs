# Example — Project minh họa tài liệu Web

Project tĩnh (HTML/CSS/JS) dùng để **test và học** theo bộ tài liệu Web: CSS, Layout (Flexbox/Grid), Responsive, Accessibility.

## Chạy project

**Cách 1:** Mở trực tiếp file `index.html` trong trình duyệt.

**Cách 2:** Dùng server tĩnh (tránh CORS khi cần):

```bash
npx serve .
# hoặc
python3 -m http.server 8080
```

Sau đó mở http://localhost:3000 (hoặc 8080).

## Nội dung minh họa

- **CSS:** variables, box model, selectors
- **Layout:** Flexbox (`.flex-demo`), Grid (`.grid-demo`)
- **Responsive:** mobile-first, media query
- **Accessibility:** semantic HTML, `aria-*`, label + id, focus visible

Đọc kèm các file trong [../](..): `01-css-fundamentals.md`, `02-css-layout-flexbox-grid.md`, `04-responsive-web-design.md`, `06-accessibility-a11y.md`.
