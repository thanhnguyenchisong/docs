## 📝 Yêu Cầu Trước Khi Bắt Đầu

- **React căn bản**: JSX, components, props, state, hooks
- **JavaScript ES6+**: destructuring, modules, async/await
- **TypeScript cơ bản (khuyến nghị)**: type/interface, union, generics
- **Node.js & npm**: đã cài đặt Node.js (v18+)
