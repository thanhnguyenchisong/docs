# Example — Project minh họa Angular

Ứng dụng Angular hoàn chỉnh, chạy được để **test và học** theo tài liệu Frontend (TypeScript, Components, Routing, Forms, RxJS, NgRx, v.v.).

## Chạy project

```bash
# Trong thư mục này (example)
npm install
npm start
```

Mở http://localhost:4200

## Build production

```bash
npm run build:prod
```

## Nội dung minh họa

- TypeScript, Components & Templates, Directives, Pipes
- Services & DI, Routing, Guards, Resolvers
- Forms, HTTP Client, RxJS
- NgRx (store, effects, selectors)
- Material UI, animations

Đọc kèm tài liệu trong [../](..) và các file `01-*.md` … `15-*.md` của bộ Frontend.
